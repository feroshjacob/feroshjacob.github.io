If coding agents are doing more of the coding, what are engineers doing?

I think the answer is not less work. It is different work.

In the final part of my application generation series, I argue that as implementation gets cheaper, software work moves toward mission, judgment, validation, integration, accountability, and helping more organizations use software well.

That matters because small businesses and nonprofits have always had software problems. They just did not always have software budgets.

Application generation may change that.
